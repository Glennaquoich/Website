# Glennaquoich Website

A simple, stylish static site for the Glennaquoich music production company.

## Quick preview

Open `index.html` in your browser to preview the site locally.

## GitHub Pages hosting

1. Create a new GitHub repository and push these files to the `main` branch.
2. In GitHub, go to **Settings → Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Choose **main** and **/ (root)**, then save.
5. Wait for GitHub Pages to publish your URL.

## Update releases

In `index.html`, replace the release titles and update each platform link with the correct URLs.
